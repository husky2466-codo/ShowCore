## Instructions

1. **Review the current session** - Look at what was discussed, decided, and accomplished
2. **Read the current CLAUDE.md** file at `C:\Users\myers\CLAUDE.md`
3. **Update the following sections:**
   - **Learned Preferences** - Add any new user preferences discovered
   - **Ongoing Projects** - Update project status, add new projects
   - **Tools & Apps** - Document any new tools created
   - **Session Memory** - Add a new dated entry with:
     - Key topics discussed
     - Decisions made
     - Problems solved
     - Issues encountered

4. **Keep it concise** - Use bullet points, no fluff
5. **Preserve existing content** - Don't delete previous session memories (keep last 15 sessions, prune older ones into summary)
6. **Show the user** - Display what was added/changed

After updating, confirm the changes were saved.
