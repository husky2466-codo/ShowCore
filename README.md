# ShowCore Export Package

This package contains the complete design specifications, components, and implementation guides for ShowCore - a technician discovery marketplace for the AV industry.

## Quick Start

### Option 1: One-Shot Implementation
Give an AI coding agent the following files:
1. `prompts/one-shot-prompt.md` - Copy this prompt
2. `product-overview.md` - For context
3. `instructions/one-shot-instructions.md` - All implementation details

### Option 2: Incremental Implementation
Build section by section:
1. Start with `instructions/incremental/01-foundation.md`
2. Then `instructions/incremental/02-shell.md`
3. Continue with each section in order
